WHAT IS WEB?

* The internet is a massive network of networks.

- It connects millions of computers together globally having a network in which any computer can communicate with another computer as long as they are both connected to the internet.

* The world wide web (WWW): is a way of accessing information over a medium of the internet.

- It is an information-sharing model that is built on top of the internet (A branch of connected computers that have web).

* A browser: is a program that allows you to interact and view contents of the web.(Application programs dedicated for web browsing e.g Opera, Firefox, Chrome, Inter explorer etc are called browsers.

* The very first page of a website is called homepage.

* Clients: Are devices (e.g laptop, tablets, mobile phone etc) for which browsers are used are referred to as CLIENTS.

* Servers: When a computer is connected to the internet and spinning out a website, we call the computer a SERVER, because it is serving the website.

* The basic code for website creation are; HTML, CSS & Javascript.

Examples of Markup Languages

* SGML: Standard Generalised Markup Language.

* HTML: HyperText Markup Languge.

* XML: eXtensible Markup Language.

* XHTML: eXtensible HyperText Markup Language.

* DHTML: Dynamic HyperText Markup Language.

* WML: Wireless Markup Language.

SGML,HTML and XML are the most important markup languages.

* SGML because it is the parent language of both HTML and XML,

* HTML because it is the current language of the web, and

* XML because it is the future language of the web.
